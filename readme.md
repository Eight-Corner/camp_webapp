###   Description

Nuxt.js 라이브러리로 변경하려는 이유는 아무래도 구성중인 레이아웃 구조가 기본 Vue 방식보다 

Nuxt의 특징들을 살려서 레이아웃을 이룰 수 있을 것 같다.

특히 Nuxt에서 지원해주는 Router 기능과 import 등 Vue.js 사용때의 수고를 덜어주는 점에서 작업에 용이할 것 같음.

SSR 방식을 사용하여 구상중인 기능들을 개발해볼 예정이다.
